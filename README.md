# 粤港澳大湾区数据要素流动多元统计分析

## 📌 项目概述

本项目基于粤港澳大湾区11个城市的2019-2023年面板数据，运用多元统计分析方法，从数据要素流动、基础设施支撑、经济发展水平、科技创新环境等多个维度，对湾区城市的数字经济发展水平进行综合评价与分类研究。

**研究目标：**
1. 构建大湾区数字经济竞争力评价指标体系
2. 揭示数据要素流动的时空格局与影响因素
3. 对湾区城市进行科学分类并分析其发展特征
4. 提出差异化政策建议，促进区域协同发展

**技术路线：** 描述性统计 → 相关性分析 → 主成分分析 → K-means聚类 → 判别分析 → 可视化呈现

## 📊 数据说明

### 数据文件：`main_data_advanced.csv`
包含2019-2023年粤港澳大湾区11个城市的面板数据：

**核心指标：**
- 经济规模：`GDP_亿元`
- 数据流动：`跨境数据传输总量_TB`、`API调用次数`、`数据交易额_亿元`
- 基础设施：`数据中心机架数`、`5G基站数量`、`互联网国际出口带宽_Gbps`
- 科技创新：`研发经费投入_亿元`、`专利授权数_件`、`高新技术企业数_家`
- 数字产业：`数字经济核心产业增加值_亿元`、`数字经济占GDP比重_%`

### 数据预处理
1. **缺失值处理**：均值插补
2. **异常值检测**：3σ原则与箱线图识别
3. **数据标准化**：Z-score标准化（消除量纲影响）

## 📈 分析模块说明

### 1. 描述性统计分析
- 时间趋势分析：绘制跨境数据传输总量年均变化趋势
- 空间分布分析：各城市研发投入对比柱状图
- 城市差异分析：多指标分组可视化

### 2. 相关性分析
- 相关矩阵热力图：展示关键变量间相关系数
- 偏相关分析：控制城市经济规模（GDP）后，识别变量间纯净关系

### 3. 降维分析（主成分分析）
- KMO与Bartlett球形检验：评估数据是否适合降维
- 主成分提取：依据特征值>1原则提取主成分
- 方差解释率分析：展示各主成分贡献率
- 载荷矩阵分析：解释各主成分实际含义

### 4. 聚类分析（K-means）
- 最优聚类数确定：轮廓系数法、肘部法则
- 聚类执行：基于主成分得分进行聚类
- 聚类结果可视化：主成分空间中的城市分布
- 类别特征对比：计算各类别关键指标均值

### 5. 判别分析（LDA）
- 判别函数构建：基于主成分得分的线性判别
- 分类效果评估：混淆矩阵、准确率计算
- 特征重要性分析：识别关键分类变量

### 6. 动态演化分析
- PC1得分动态演化：展示各类别城市随时间变化趋势
- 复合年增长率分析：计算各城市数据流量CAGR
- 发展路径图：可视化代表性城市的发展轨迹

## 📊 主要分析结果

### 1. 城市分类结果（2023年）
| 类别名称 | 城市数量 | 代表性城市 | 主要特征 |
|----------|----------|------------|----------|
| 创新引领型 | 1 | 深圳 | 研发投入极高、数字经济占比领先 |
| 枢纽支撑型 | 2 | 广州、香港 | 跨境数据流量大、基础设施完善 |
| 外围发展型 | 8 | 东莞、佛山、珠海等 | 整体指标相对落后、发展不均衡 |

### 2. 关键发现
- **马太效应明显**：核心城市优势持续扩大，外围城市追赶难度大
- **数据流量与创新投入分离**：深圳创新强但数据流量相对较低
- **基础设施追赶较快**：5G基站数量差距小于经济与创新差距
- **区域协同不足**：城市间数据流动存在制度与技术障碍

### 3. 政策建议
- **深圳**：建设"数据自贸区"，试点跨境数据流动特殊政策
- **广州-香港**：建立数据合作示范区，统一标准与接口
- **外围城市**：结合产业特色实施差异化追赶策略
